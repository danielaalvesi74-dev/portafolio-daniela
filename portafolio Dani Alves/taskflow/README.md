TaskFlow

TaskFlow es una aplicación web sencilla para organizar tareas.
La desarrollé como parte de mi práctica de JavaScript para aplicar los conceptos vistos en clase.

La idea fue construir algo funcional, claro y bien estructurado, sin hacerlo más complejo de lo necesario.
¿Qué hace?

Con TaskFlow puedes:

Crear tareas con una descripción y una fecha opcional

Marcar tareas como completadas

Eliminar tareas

Filtrar por estado (todas, pendientes o completadas)

Buscar tareas en tiempo real

Guardar automáticamente las tareas en el navegador

También incluye:

Un botón de tareas de ejemplo (Demo)

Una simulación de sincronización con una API pública (ejercicio académico)
Cómo usarlo

Solo necesitas abrir el proyecto con Live Server en VS Code y usarlo desde el navegador.
